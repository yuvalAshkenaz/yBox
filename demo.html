JavaScript parameters

code : When calling to yBox from your Java Script, you can write your HTML code
<script> yBox({ code: '<div class="my-class">Some text</div>' }); </script>
self : <a href="#popup" class="yBox my-link">The link</a>
<script> yBox({ self: jQuery('.my-link') }); </script>
yBoxClass : Added to <div class="yBoxFrame"> in yBox window
<script> yBox({ yBoxClass: 'my-class' }); </script>
focus : Focus the element you want when yBox opens
<script> yBox({ focus: '.focus-class' }); </script>
url : When calling to element from java script.
<div style="display:none;">
	<div id="my-popup">
	...
	</div>
</div>
 <script> yBox({ url: '#my-popup' }); </script>

Button Attributes

data-ybox-class : Add class to yBox popup
<button type="button" href="#popup" data-ybox-class="my-class" class="yBox">
	Click Here
</button>
data-ybox-alt : Add Alt to yBox image
<a href="my-img.jpg" data-ybox-alt="My Alt" class="yBox">
	<img src="my-img.jpg" alt=""/>
</a>
data-ybox-title : Add title to image or iframe
<a href="https://www.my-iframe.com" data-ybox-title="My Title" class="yBox">
	Click Here
</a>

Output:
<iframe src="https://www.my-iframe.com" title="My Title"></iframe>
data-ybox-headline
data-ybox-headline-class 		: Add headline (H2) to image or iframe
<a href="my-img.jpg" data-ybox-headline="My Headline" data-ybox-headline-class="my-class" class="yBox">
	Click Here
</a>

Output:
<h2 id="ybox-iframe-headline" class="my-class"> My Headline </h2>
<iframe src="https://www.my-iframe.com" id="yBoxIframe" class="yBoxIframe"></iframe>
data-ybox-group : Switch images
<a href="my-img.jpg" data-ybox-group="my-group" class="yBox">
	<img src="my-img.jpg" alt=""/>
</a>

URL Attributes

https://domain.com?msg=Some text : msg parameter in url pop the yBox with your text.
https://domain.com?ybox-url=https://www.domain.com
https://domain.com?ybox-headline=Some Title 		: Pop iframe with the url.
https://domain.com?ybox-id=my-popup : Call to element from URL address
<div style="display:none;">
	<div id="my-popup">
	...
	</div>
</div>
https://domain.com?ybox-class=my-popup-class : add class to yBox window from URL address


Callbacks

function beforeYboxOpen(self){ ... }; : Calling the function before opening yBox
self = The yBox button -> if no button then it return the div
function afterYboxOpen(self){ ... }; : Calling the function after opening yBox
self = The yBox button -> if no button then it return the div
function beforeYboxClose(self){ ... }; : Calling the function before closing yBox
self = The yBox button -> if no button then it return the div
function afterYboxClose(self){ ... }; : Calling the function after yBox closed
self = The yBox button -> if no button then it return the div
